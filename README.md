# Full Stack Contact Form

A Full Stack Contact Form application built using React.js, Node.js, and Express.js with API integration and form validation.

## Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

## Features
- User contact form
- Form validation
- API integration
- Full stack architecture
- Responsive UI

## Project Structure

```bash
frontend/
backend/
```

## Installation

### Clone Repository

```bash
git clone https://github.com/keerthanaarige/full-stack-contact-form.git
```

### Install Dependencies

```bash
npm install
```

### Start Frontend

```bash
npm start
```
